# C Shell
A fully functional Interactive Unix Bash like Shell in C.
Features:

1. Signal Handling
2. Multi Threading
3. Process Creation
4. Background and Foreground Processes
5. User defined and built-in commands
6. I-O Redirection
7. Piping

HOW TO USE 
1. Compile the file LinuxShell.c with gcc
2. Run ./a.out
